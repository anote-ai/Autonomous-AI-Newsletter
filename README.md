### To run the locally you should:

#### Clone this repo to your local machine

```
git clone https://github.com/nv78/Anote-AutonomousNewsletter.git
```

#### To run the Backend
```
cd server
cd python
npm run start
```

#### To run the Frontend
```
cd my-app
npm install
npm start
```
